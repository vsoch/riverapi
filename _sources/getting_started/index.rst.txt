===============
Getting Started
===============

The River API provides a standard client and specification for interacting with a `river <https://riverml.xyz>`_ online machine learning server. 
We provide documentation here, and you can also find `examples <https://github.com/vsoch/django-river-ml/tree/main/examples>`_
for the Django River ML plugin.

If you have any questions or issues, please `let us know <https://github.com/vsoch/riverapi/issues>`_
